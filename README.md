# ST-Leaderboards

**ST-Leaderboards** is a high-performance, leaderboard system for modern Minecraft servers (Spigot, Paper, Purpur). Built by **Septy Studios** to be lightweight, reliable, and fully customizable.

---

### **Key Features**

* **Zero Lag:** All data sorting and fetching is handled **asynchronously** to keep your TPS at 20.0.
* **PlaceholderAPI Support:** Works with any statistic that has a placeholder.
* **Full Customization:** Every message, prefix is configurable. **No hardcoded text.**

---

### **Commands & Permissions**

- **/leaderboard create <id> <type>** | Create a new leaderboard | `st-leaderboards.admin` |
- **/leaderboard delete <id>** | Remove an existing leaderboard | `st-leaderboards.admin` |
- **/leaderboard move <id>** | Move an existing leaderboard | `st-leaderboards.admin` |
- **/leaderboard reload** | Reload all configuration files | `st-leaderboards.admin` |

---

### **Quick Setup**

1.  Drop the `ST-Leaderboards.jar` into your `/plugins/` folder.
2.  Restart the server to generate the **config.yml**.
3.  Create your first leaderboard using `/leaderboard create`.
